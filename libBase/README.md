# libBase仓库

### 宏

- 用于调试 ```HINT```  (在任意行添加HINT会打印HINT所在文件以及行数，形成类似于断点的功能)
- 数学 ```PI```

### 类

- 帧率控制 ```Rate```
- 带时间戳图像帧 ```MatWithTime```
- 云台位姿角 ```GimbalPose```

### 函数

- 获取当前时间戳 ```now()```
- 获取当前时间戳（微秒为单位） ```getTimeUSecNow()```
- 终端按键监听 ```monitorKeyboard()```
- 彩色化终端输出 ```_colors()```
